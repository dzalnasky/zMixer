# zMixer
zMixer is a 0-4GHz IF to 5-10.4GHz RF mixer, with dedicated LO chip generating 0-6.4GHz. Programmable via USB-C.
# Key Components
* Texas Instruments LMX2572 Frequency Synthesizer
  * Drop in replacements include the LMX2592, LMX2594, and LMX2595 for higher LO frequency generation
* Microchip MCP2210 USB to SPI Interface
* Analog Devices HMC220B RF Mixer
  * Drop in replacement includes HMC218B

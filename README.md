# zMixer
zMixer is a USB-C programmable, 0-4GHz IF to 5-10.4GHz RF upconverting/downconverting RF mixer with dedicated LO chip generating 0-6.4GHz. Perfect for radios with operating frequencies up to 4GHz that want to work with higher frequency bands. 
# Key Components
* Texas Instruments LMX2572 Frequency Synthesizer
  * Drop in replacements include the LMX2592, LMX2594, and LMX2595 for higher LO frequency generation **(May require different RF mixer IC)*
* Microchip MCP2210 USB to SPI Interface
* Analog Devices HMC220B RF Mixer
  * Drop in replacement includes HMC218B
